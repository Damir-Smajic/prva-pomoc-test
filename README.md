# Neira testovi - GitHub Pages split verzija

Ovo je split verzija testa za GitHub Pages.

## Struktura

- `index.html` - glavna stranica
- `styles.css` - stilovi
- `app.js` - logika testa
- `data/` - pitanja i tačni odgovori u JSON fajlovima
- `assets/` - slike za znakove i saobraćajne situacije

## GitHub Pages

Uploaduj sve fajlove i foldere iz ovog ZIP-a u root repozitorija.
`index.html` mora ostati u rootu.

## Važna korekcija

Za kategoriju `Saobraćajne situacije / Oblast III`, pitanje 98 je reanalizirano iz PDF-a i u JSON-u je postavljeno kao tačno: `1` i `4`.
